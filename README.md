# MARK v8
MARK v8 is a cloud-first autonomous AI operations system designed to automate business workflows. It integrates mobile and desktop agents with WhatsApp, Google Sheets, Email, and Calendar to execute tasks with an approval-based workflow.
## 🚀 Key Features
- 🌐 Cloud-first architecture with always-on execution
- 📱 Mobile dashboard for monitoring and approvals
- 💻 Desktop worker for local automation and browser control
- 🤖 Hierarchical subagent system (CEO, Planner, Router, Execution Agents)
- 📊 Google Sheets, Gmail, and Calendar integrations
- 💬 WhatsApp automation with approval workflow
- 🧠 Learning engine (Watch-to-Learn, Search-to-Skill, Pattern Detection)
- 🔔 Push notifications and real-time updates
- 🔐 Role-based authentication and security
## 📦 Deployment
1. Clone the repository.
2. Deploy the cloud backend using Docker.
3. Install the Android APK.
4. Register the Desktop Worker.
5. Configure integrations via the dashboard.
